# Focus Gate – Mindful Browsing Extension

## Overview

Focus Gate is a productivity-focused browser extension designed to reduce impulsive browsing and improve digital self-control.

Instead of instantly blocking websites, Focus Gate introduces a short reflection period before accessing distracting content, helping users make more intentional decisions about their online behavior.

The extension aims to reduce mindless scrolling and encourage conscious internet usage without completely restricting access.

---

## Features

* Customizable list of distracting websites
* Reflection screen before accessing blocked sites
* Countdown timer to prevent impulsive clicks
* Option to proceed or return to productive work
* Clean and distraction-free user interface
* Lightweight browser extension with minimal performance impact

---

## How It Works

When a user attempts to visit a distracting website:

1. The extension intercepts the request.
2. A focus screen appears with a short countdown timer.
3. The user is encouraged to reconsider the visit.
4. After the timer expires, the user can either:

   * Continue to the website
   * Return to their work

This creates a small friction point that reduces habitual and unconscious browsing behavior.

---

## Problem Statement

Many website blockers rely on strict restrictions that users often disable or bypass.

Focus Gate takes a different approach by leveraging behavioral psychology and intentional friction to interrupt impulsive actions while preserving user autonomy.
